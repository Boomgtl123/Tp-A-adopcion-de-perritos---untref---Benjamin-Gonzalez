# Desafío 1 - Adopta un perrito (Vite + React)

## Cómo correrlo

```bash
npm install
npm run dev
```

## Estructura
- `src/components/Header.jsx` (título por **prop** desde `App.jsx`)
- `src/components/MyCard.jsx` (recibe `image`, `name`, `description` + tag)
- `src/components/Tags.jsx` (Badge de **Bootstrap**, recibe `text` y `color`)
- `src/components/Footer.jsx` (texto del footer)
- `src/styles.css` (estilos para parecerse al diseño de referencia)

## Nota
Las imágenes se cargan desde URLs públicas (Unsplash) para que no tengas que subir archivos.
